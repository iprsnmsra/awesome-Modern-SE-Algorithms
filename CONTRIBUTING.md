
# 🔍Contributing to Awesome Modern SE Algorithms

Welcome! If you have found a more optimal approach, a better real-world example, or want to add a missing language implementation, your pull requests are highly encouraged.

## How to Contribute
1. **Fork the Repository**: Create your own copy of the project.
2. **Follow the Single-File Rule**: All code and CI/CD tests must be contained within a single file for each language.
3. **Run Tests Locally**: Use `run_tests.ps1` (Windows) or `run_tests.sh` (Mac/Linux) to ensure your code passes.
4. **Submit a Pull Request**: Explain your optimization. If your code drops the Time Complexity or handles edge cases better, it will be merged!
